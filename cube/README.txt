-------------------------------------
ruby sim.rb 
-------------------------------------

1. It will draw a cube in 3D space
- with the center at [0, 0, 0], and 
- with the dimensions 100 * 100 * 100

2. Apply perspective
'P' : applies/undoes the perspective

3. Rotate the cube 
'J' : rotates the cube by 20 degrees along +x
'K' : -x
'H' : +y
'L' : -y
'A' : +z
'S' : -z

4. For now, there is only one coordinate system :P
